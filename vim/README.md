### Cheat sheet
* gt - Next tab
* gT - Previous tab
* ,h - Toggle showing hidden chars
* ^h ^j ^k ^l - Navigate splits
* ,s - Sorts selected lines
* Q - Wraps paragraph
* ,l - Redraws the screen and gets rid of any hilighted search results
* ,z - Toggles centered cursor
* ,c - Toggle hilight current column
* ,e - EasyMotion
* ^n - Multiple Cursors
* ^p - find files
* ^f ^b - ctrlp modes
* gc - comment
